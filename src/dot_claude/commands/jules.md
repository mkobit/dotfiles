---
description: Interact with the Jules API
argument-hint: "[list|create|view]"
---

This command allows you to interact with the Jules API using a CLI tool.

It supports the following subcommands:
- `list`: List all active sessions.
- `create`: Create a new session.
- `view <session_id>`: View details and activities of a specific session.

To run the interactive CLI:
```bash
jules
```

To list sessions:
```bash
jules list
```

To create a session:
```bash
jules create
```

To view a specific session:
```bash
jules view <session_id>
```
