# AI skill deployment

Skills are deployed as static files into each tool's discovery directory (`~/.claude/skills/`, `~/.gemini/antigravity-cli/skills/`).
There is no marketplace, plugin CLI, or symlinking — chezmoi owns the full lifecycle.

Two kinds of skills share one catalog: `src/chezmoi/.chezmoidata/ai/skills.toml`.

## Upstream skills

Pinned GitHub archives, one chezmoi external per (skill, tool), generated by `src/chezmoi/.chezmoiexternals/ai-skills.toml.tmpl`.
Each archive is verified against its pinned sha256, then piped through `src/python/skill_filter` to extract a single skill directory.

To add a source and select skills:

```toml
[ai.skills.external.superpowers]
repo = "obra/superpowers"
ref = "<full commit sha>"
sha256 = "<sha256 of the downloaded archive>"

[ai.skills.external.superpowers.skills]
brainstorming = "present"
```

Each source downloads from an archive host (`host`, default `"github"`) declared under `[ai.skills.hosts.<name>]` with a `url_format` substituting `{repo}` and `{ref}`.
Overlays can repoint a host at a proxy, add internal hosts (for example GitHub Enterprise), or set `url` on a source for a one-off published archive.

To compute the pin for a new ref:

```sh
curl -sL https://codeload.github.com/<owner>/<repo>/tar.gz/<commit-sha> | sha256sum
```

To remove a skill, set it to `"absent"` — never delete the key, which would orphan the installed copy.
The `.chezmoiremove.tmpl` files in `dot_claude/` and `dot_gemini/` prune absent skills on apply.

## Authored skills

Full directory trees under `src/ai/skills/<name>/`, listed in `[ai.skills.authored]`, deployed as copies by `src/chezmoi/.chezmoiexternals/ai-authored-skills.toml.tmpl`.
New files inside a present skill are picked up automatically; edits propagate on the next `chezmoi apply`.
To add a skill: create `src/ai/skills/<name>/SKILL.md` and add `<name> = "present"` to the catalog.

## Overlay composition

Config `[data.*]` values take per-leaf precedence over the catalog:

```toml
# Route GitHub downloads through a proxy.
[data.ai.skills.hosts.github]
url_format = "https://artifacts.corp/github/{repo}/tar.gz/{ref}"

# Deselect a skill on this machine.
[data.ai.skills.external.superpowers.skills]
brainstorming = "absent"
```

Overlay-only sources go in an overlay `.chezmoidata` file under distinct `[ai.skills.external.<name>]` keys.

## Verify and test

```sh
chezmoi diff
chezmoi apply ~/.claude/skills ~/.gemini/antigravity-cli/skills
uv run pytest src/python/skill_filter
```
